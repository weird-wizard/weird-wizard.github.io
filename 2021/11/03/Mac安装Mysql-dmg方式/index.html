Mac安装Mysql(dmg方式)

安装环境

  • 机器：2020款MacBook Pro（M1版）
  • 系统：macOS Monterey 12.0.1

1、下载Mysql包&安装

Mysql官网：https://www.mysql.com/

1.1 选择MySQL Community Server

进入官网，在页面的最底部，Downloads下点击MySQL Community Server

1.2 选择版本安装包&下载

因为本机是M1处理器，所以选择arm版的dmg安装包，点击按钮。

这里下载的是最新版的Mysql，如果想下载其他版本，可以点击Archive按钮。

点击No thanks, just start my download.进行下载。

1.3 安装

安装和大多是mac安装包一样，只需要配置其密码即可，其他不用更改。
